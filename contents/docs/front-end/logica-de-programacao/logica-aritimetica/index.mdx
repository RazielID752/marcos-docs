"Lógica" e "aritmética" são conceitos distintos, mas ambos são fundamentais em programação, especialmente em linguagens como C.

Lógica:
Em programação, a lógica se refere à maneira como os programas são estruturados para tomar decisões ou realizar operações com base em condições. Ela envolve o uso de instruções condicionais (como if, else, switch) e operadores lógicos (como AND, OR, NOT) para controlar o fluxo do programa. A lógica é essencial para a construção de algoritmos e para criar programas que podem tomar decisões com base em certas condições.

Aritmética:
A aritmética está relacionada aos cálculos matemáticos realizados dentro de um programa. Isso inclui operações matemáticas básicas, como adição, subtração, multiplicação, divisão e outras operações mais complexas, dependendo do contexto do programa. Em linguagens de programação como C, os operadores aritméticos são usados para realizar cálculos numéricos, e eles incluem símbolos como +, -, *, /, % (para resto da divisão), entre outros.

Na prática da programação em C, a lógica é usada para controlar o fluxo do programa, enquanto a aritmética é usada para manipular valores numéricos e realizar cálculos matemáticos. Ambas são habilidades cruciais para um programador dominar, pois são a base para resolver problemas e criar soluções eficientes em código
