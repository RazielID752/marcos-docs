## O que são Estruturas de Dados Homogêneas?

As **estruturas de dados homogêneas** são aquelas que armazenam elementos do mesmo tipo de dados de forma contígua na memória. Isso significa que todos os elementos dentro da estrutura têm o mesmo tamanho e estão armazenados em posições adjacentes. Essas estruturas são muito utilizadas em programação para organizar e manipular dados de maneira eficiente.

### Exemplos Comuns

1. **Vetores (ou Arrays):**
   Um **vetor** é uma estrutura de dados que armazena uma coleção de elementos do mesmo tipo. Cada elemento é acessado através de um índice. O primeiro elemento de um vetor geralmente tem o índice `0`, e o último elemento tem o índice `n-1`, onde `n` é o número de elementos no vetor.

2. **Matrizes (ou Arrays Multidimensionais):**
   Uma matriz é uma extensão dos vetores, permitindo o armazenamento de dados em várias dimensões (como tabelas ou grades).

## Exemplo de Vetor em Linguagem C

Aqui está um exemplo simples de como declarar, inicializar e acessar elementos de um vetor em C:

```c
#include <stdio.h>

int main() {
    // Declarando e inicializando um vetor de inteiros com 5 elementos
    int vetor[5] = {1, 2, 3, 4, 5};

    // Iterando sobre os elementos do vetor e imprimindo cada um
    for (int i = 0; i < 5; i++) {
        printf("Elemento na posição %d: %d\n", i, vetor[i]);
    }

    return 0;
}
```
