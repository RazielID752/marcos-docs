HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar páginas na web. Ele define a estrutura básica de um site por meio de elementos que organizam o conteúdo, como títulos, parágrafos, imagens, links e tabelas.

## Características do HTML
- **Linguagem de marcação:** HTML não é uma linguagem de programação, mas sim uma linguagem de marcação que define a estrutura do conteúdo.
- **Base da Web:** Todos os sites da internet utilizam HTML em sua composição.
- **Trabalha junto com CSS e JavaScript:** HTML define a estrutura, CSS cuida do estilo e JavaScript adiciona interatividade.

## História do HTML
O HTML foi criado por **Tim Berners-Lee** em 1991. Desde então, passou por várias versões, sendo a mais recente o **HTML5**, que trouxe novos elementos semânticos, suporte a multimídia e APIs avançadas.

## Exemplo de um documento HTML básico
```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Primeiro HTML</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Esse é um documento HTML básico.</p>
</body>
</html>
```
