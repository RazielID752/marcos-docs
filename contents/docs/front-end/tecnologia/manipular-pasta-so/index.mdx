#  

## Windows  

### Explorador de Arquivos  

O **Explorador de Arquivos** é a principal ferramenta do Windows para gerenciar pastas e arquivos. Com ele, você pode:  
- Criar novas pastas;  
- Renomear arquivos e pastas;  
- Copiar, mover ou excluir itens.  

### Atalhos e Renomeação  
- Para criar um atalho de um arquivo ou pasta:  
  1. Clique com o botão direito sobre o item;  
  2. Selecione a opção **"Criar atalho"**.  
- Para renomear: clique uma vez no nome do arquivo ou pasta, aguarde um instante e clique novamente para editar.  

### Comandos de Teclado  
Alguns atalhos úteis para manipular arquivos no Windows:  
- **`Ctrl + C`** Copiar;  
- **`Ctrl + X`** Recortar;  
- **`Ctrl + V`** Colar;  
- **`Delete`** Excluir.  

## Mac OS  
### Finder  
No macOS, o **Finder** é a ferramenta equivalente ao Explorador de Arquivos do Windows. Ele oferece funcionalidades como:  
- Criar pastas e arquivos;  
- Copiar, mover ou excluir itens;  
- Organizar e buscar documentos de forma prática.   

## Linux  
### Gerenciador de Arquivos  
No Linux, o gerenciador de arquivos pode variar de acordo com o ambiente gráfico utilizado. Alguns dos mais comuns são:  
- **Nautilus** (GNOME);  
- **Dolphin** (KDE);  
- **Thunar** (XFCE).  

Com o gerenciador de arquivos, você pode:  
- Criar, renomear, copiar, mover e excluir arquivos e pastas;  
- Arrastar e soltar itens para organizá-los;  
- Acessar drives externos com facilidade.  

### Terminal  
O terminal é uma ferramenta poderosa no Linux para manipular arquivos e pastas. Alguns comandos básicos incluem:  
- **`ls`**: Lista arquivos e pastas no diretório atual;  
- **`cd [nome_do_diretorio]`**: Acessa um diretório;  
- **`mkdir [nome_da_pasta]`**: Cria uma nova pasta;  
- **`rm [nome_do_arquivo]`**: Remove arquivos;  
- Use **`rm -r [nome_da_pasta]`** para remover pastas e seus conteúdos.  
- **`cp [origem] [destino]`**: Copia arquivos ou pastas;  
- **`mv [origem] [destino]`**: Move ou renomeia arquivos e pastas.  


> **Observação**: Com essas ferramentas e atalhos, manipular pastas e arquivos no Windows e macOS torna-se simples e eficiente.
